◆ readUserGuess()

char readUserGuess ( FILE *  pLog,
char *  wordToGuess 
)

Scans and checks user guess and returns it to main.

Parameters
*pLogLog file
*wordToGuessSecret Hangman word

Asks the user to enter a letter and capitalizes input. Writes current time into log file.

4 options:

  1. If user manages to guess the whole word, they instantly win the round and the game restarts.
  2. If user inputs more than 1 character and fail to guess the secret word it's "Game over".
  3. If user input is 1 alphabetic character, the entered character is returned.
  4. If user input is 1 non-alphabetic character a message appears and the function restarts.
    Returns
    guess[0] = 0 with option 1. and 2. - game restarts in main()
    guess[0] if user enters an alphabetic character
    checkUserGuess() if user enters non-alphabetic character
